6. NEVER：以非事务方式执行，如果当前存在事务，则抛出异常。

7. NESTED：如果当前存在事务，则创建一个事务作为当前事务的嵌套事务来执行；如果当前没有事务，则该取值等价于REQUIRED。

####  #证明费马定理 <span class="gpt-item">bili_287913366@05:41:29 (还有1次)</span>



费马定理：如果n是一个大于2的自然数，那么对于任意整数a，都有a^n ≡ a (mod n)。

证明：

假设n是一个大于2的自然数，令a=1，则有1^n ≡ 1 (mod n)，

因为n&gt;2，所以1&lt;n，这时有：

1^n−1 ≡ 0 (mod n)

又因为a^n−1−1=(a−1)(a^n−1+a^n−2+⋯+a+1)，

所以有：

(a−1)(a^n−1+a^n−2+⋯+a+1) ≡ 0 (mod n)

由于a^n−1+a^n−2+⋯+a+1≡n (mod n)，

所以有：

(a−1)n ≡ 0 (mod n)

即a^n ≡ a (mod n)

以上证明了当n&gt;2时，对于任意整数a，都有a^n ≡ a (mod n)，也就是费马定理。

      let prevArgs: any[] = []

    let temp, gap = 1;
    while(gap &lt; len/3) {

    }
    for(gap; gap &gt; 0; gap = Math.floor(gap/3)) {
        for(let i = gap;i &lt; len; i++) {
            temp = arr[i];
            let j = i - gap;
            while(j &gt;= 0 && arr[j] &gt; temp) {
